Experience

SpellStrikeXIV: A 2D Side-Scrolling Typing Combat Game (2022)

ProjectOverview:
SpellStrikeXIV is a 2D side-scrolling game that challenges players to type accurately and quickly to execute skills and combat enemies. The gameplay revolves around defeating enemies to earn points, with the game ending when the player's health reaches zero. At the end of each round, the player's score is displayed. If the score surpasses all previous records, it is recorded as the new High Score.

Vansenger Counter: Smart Passenger Counting System for Vans (2023)

ProjectOverview:
Vansenger Counter is a project that utilizes the Arduino WIFI R4 board to display the number of passengers in a van. If the van reaches full capacity, the system will indicate "Full." Otherwise, it will show the number of available seats. Additionally, users can access key van details via a web interface, including: Driver information Van route name Departure and destination points This system helps improve passenger management and enhances convenience for both drivers and passengers.
